SunFounder PiCar-4wd Car Kit for Raspberry Pi
===================================================

About PiCar-4wd Kit
------------------------

PiCar-4wd is a cost-effectively developmental smart car kit of Raspberry Pi, suitable for the Raspberry Pi 3 model B/B+ and 4 Model B. With the grayscale module, ultrasonic module, Photo-interrupter sensor module, the car can realize obstacle avoidance, line following, object following, cliff detection, velocity measurement, mileage calculation and so on. 
Besides, the kits comes with operational examples of Web terminal, a rookie-friendly design. We also provide the original python code for in-depth learning.
